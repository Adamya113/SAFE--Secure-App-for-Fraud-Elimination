# To run the code:

## 1. Make sure you have nodejs installed on your PC.

## 2. Starting dashboard, write in terminal
   </br> a. npm install
   </br> b. cd react-admin
   </br> c. npm start

## 3. Starting backend, write in a new parallel terminal
   </br> a. pip install requirements.txt
   </br> b. cd backend
   </br> c. python app.py
